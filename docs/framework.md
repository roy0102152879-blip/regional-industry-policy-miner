# Framework

## Why This Exists

Local industry research often fails because it treats every policy keyword as real industrial strength. Many cities mention "new energy", "semiconductors", "AI", "low-altitude economy", or "new materials", but the meaning varies widely:

- Some have real production scale and listed-company clusters.
- Some have only project announcements.
- Some have strong research institutions but weak commercialization.
- Some have policy ambition without local carrying capacity.

This framework turns policy language into a structured evidence map.

## Two Modes

### Regional Blind Scan

Start from a region. Ask:

- What industries appear repeatedly in official planning?
- Which industries get funds, projects, land, platforms, parks, or organizational mechanisms?
- Which industries have real local firms or project subjects?
- Which industries deserve a targeted deep dive?

This is the mode closest to the spirit of policy-text decoding research.

### Targeted Industry Deep Dive

Start from a region + industry. Ask:

- How long has the industry been supported?
- Which level of government supports it?
- How much resource has been committed?
- What is the local advantage?
- What is the cluster type?
- Which enterprises carry the industry?
- How large is it, how fast is it growing, and what target gap remains?

## Three Judgments

Keep these separate:

| judgment | question |
| --- | --- |
| Policy bet | Is the government really pushing it? |
| Local carrying | Can local actors actually carry it? |
| Commercial pre-check | Does preliminary market/financial evidence look healthy? |

The first two are the core of this framework. The third is a lightweight risk screen.
