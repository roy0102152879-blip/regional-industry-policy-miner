# Field Definitions

## Candidate Industry Scoring

| field | meaning |
| --- | --- |
| policy_target_strength | whether the industry is a primary policy target |
| tool_complexity | whether support tools go beyond slogans |
| organization_execution | whether there are targets, offices, platforms, parks, funds, or coordination mechanisms |
| local_fit | whether the industry uses local research, resources, location, firms, or industrial base |
| enterprise_project_carrying | whether real firms, projects, or platforms carry the industry |
| resource_commitment | whether money, land, funds, credit, procurement, or project investment is visible |
| continuity | whether support continues across years and policy levels |

## Enterprise Fields

| field | meaning |
| --- | --- |
| enterprise_name | firm, project company, platform, or institution |
| role | chain owner, upstream, midstream, downstream, platform, project subject |
| listed_status | listed, listed-company subsidiary, IPO/terminated, non-listed, pending |
| ticker_or_parent | ticker or listed parent |
| local_link | registration, production base, project, park, resource, service |
| scale_signal | revenue, profit, output, investment, capacity, employment, customers |
| verification | official, annual report, exchange filing, company website, authority media, media, pending |

## Quantitative Fields

| field | meaning |
| --- | --- |
| metric_name | output, enterprise count, fund size, platform investment, listed-company revenue, etc. |
| value | number and unit |
| period | year, quarter, or as-of date |
| geography_or_scope | province, city, district, park, company, platform |
| baseline_or_target | actual, historical, policy target, estimate |
| source_quality | official statistics, formal policy, listed filing, government news, media, company claim, estimate |
| caveat | target not actual, non-additive, duplicated, incomplete time series |
