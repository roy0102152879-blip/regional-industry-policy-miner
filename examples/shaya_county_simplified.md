# Example: Shaya County Regional Scan (Simplified)

This is a simplified public example. It demonstrates the kind of judgment the
skill should make for a county-level resource region.

## Short Conclusion

Shaya County should not be read as a generic "many promising industries" case.
The stronger pattern is a resource/county industrial model:

1. Cotton full-chain processing
2. Oil, gas, and natural-gas chemical processing
3. Photovoltaic sand-control and desert-related industry

Cotton and oil/gas have clearer local resource foundations. PV sand-control is
policy-relevant and spatially distinctive, but it needs careful separation
between power-generation projects, ecological governance, and sustainable local
industrial value.

## What This Case Tests

- Whether the skill can avoid turning a county scan into a fashionable-sector list.
- Whether resource endowment, local carrying capacity, and policy ambition stay separate.
- Whether the final output identifies which industries deserve a targeted Mode A report.

## Caveat

This public card omits full source tables and detailed enterprise verification.
A formal report should include official documents, project subjects, enterprise
roles, scale signals, and source URLs for every material claim.
