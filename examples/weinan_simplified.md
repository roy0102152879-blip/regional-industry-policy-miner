# Example: Weinan Regional Scan (Simplified)

This is a simplified public example. It demonstrates how a quick regional scan
should produce a ranked follow-up queue.

## Short Conclusion

Weinan's regional scan should separate new policy bets from existing commercial
bases. A useful first-pass ranking is:

1. New-energy power generation and new-energy equipment
2. Equipment manufacturing / intelligent equipment / additive manufacturing
3. New materials / modern chemicals / molybdenum-aluminum-magnesium processing
4. Specialty agriculture full-chain and food manufacturing

Specialty agriculture may have a stronger existing commercial base than some
new-policy bets, but it should not automatically outrank sectors where policy
tools, project investment, and platform construction are more explicit.

## What This Case Tests

- Whether the scan ends with a Mode A sequence instead of a flat industry list.
- Whether the skill distinguishes policy momentum from existing market base.
- Whether broad labels such as "new energy" are broken into local meanings.

## Caveat

This public card is intentionally compact. A formal run should attach verified
source links, project subjects, enterprise mapping, quantitative targets, and
counterexamples.
